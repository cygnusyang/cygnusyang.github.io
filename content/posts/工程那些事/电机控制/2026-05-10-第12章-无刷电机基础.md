## 电机类型对比

| 特性 | 有刷直流电机 | 无刷直流电机 (BLDC) |
|---|---|---|
| 换向方式 | 机械换向（电刷+换向器） | 电子换向（功率器件+控制器） |
| 寿命 | 受电刷磨损限制 | 轴承寿命为主，寿命更长 |
| 效率 | 70-80% | 85-95% |
| 维护 | 需更换电刷 | 无需维护 |
| 噪声 | 较高 | 较低 |
| 成本 | 较低 | 较高 |
| 控制复杂度 | 简单 | 复杂（需电子换向） |

## 工作原理

### 反电动势 (Back-EMF)

当电机旋转时，转子磁场切割定子绕组，产生反电动势：

```
E = k_e × ω

其中：
E = 反电动势 (V)
k_e = 反电动势常数 (V·s/rad)
ω = 角速度 (rad/s)
```

**反电动势波形分类：**

1. **梯形波 (Trapezoidal)**
   - 方波控制（六步换相）
   - 反电动势为梯形
   - 转矩脉动较大

2. **正弦波 (Sinusoidal)**
   - FOC 控制
   - 反电动势为正弦
   - 转矩平滑

## 转矩生成原理

### 力学基础

定子磁场与转子磁场的相互作用产生电磁力：

```
F = B × I × L × sin(θ)

其中：
F = 电磁力 (N)
B = 磁通密度 (T)
I = 电流 (A)
L = 导体长度 (m)
θ = 定子磁场与转子磁场的夹角
```

### 最大转矩条件

当 θ = 90° 时，`sin(θ) = 1`，获得最大力：

```
F_max = B × I × L
```

### 转矩方程

```
τ = k_t × I

其中：
τ = 电磁转矩 (N·m)
k_t = 转矩常数 (N·m/A)
I = 电枢电流 (A)
```

**关键关系**：`k_e = k_t`（国际单位制下）

## KV 值

KV 值表示电机转速与电压的关系：

```
KV = ω / V

其中：
KV = 转速常数 (RPM/V)
ω = 转速 (RPM)
V = 电压 (V)
```

**推导关系**：

```
E = k_e × ω  (反电动势)
空载时：V ≈ E
因此：ω = V / k_e
KV = 1 / k_e

单位转换：
k_e (V·s/rad) → KV (RPM/V)
KV = 60 / (2π × k_e)
```

## 换向与换相

### 六步换相（方波控制）

```
Step 1: U+ V-  W悬空
Step 2: U+   W- V悬空
Step 3: V+ W-  U悬空
Step 4: V+   U- W悬空
Step 5: W+ U-  V悬空
Step 6: W+   V- U悬空
```

**特点：**
- 每 60° 换相一次
- 每相导通 120°
- 实现简单，成本低
- 转矩脉动较大

### 连续换相（FOC 控制）

通过计算转子位置，实时调整三相电流相位，实现平滑换向。

## 传感器类型

### 1. Hall 传感器

- **类型**：霍尔效应传感器
- **输出**：数字信号（高/低）
- **精度**：60° 分辨率
- **优点**：简单、廉价、抗干扰
- **缺点**：精度低

### 2. 编码器

- **类型**：
  - 增量式
  - 绝对式
- **输出**：ABZ 相位信号
- **精度**：高（可达 12bit+）
- **优点**：高精度、分辨率可调
- **缺点**：较复杂、成本较高

### 3. 磁编码器 (AMR/霍尔 IC)

- **类型**：AS5600、MA730 等
- **输出**：PWM 或 I2C/SPI
- **精度**：中高（8-14bit）
- **优点**：无接触、寿命长、体积小

### 4. 无传感器

- **方法**：
  - 反电动势过零检测
  - 观测器（滑模、扩展卡尔曼滤波）
- **优点**：无额外硬件
- **缺点**：低速性能差、算法复杂

## 电机分类

### 按转子结构

1. **表贴式永磁 (SPM)**
   - 磁体贴在转子表面
   - 电感小，响应快
   - 凸极率 L_d/L_q ≈ 1

2. **内嵌式永磁 (IPM)**
   - 磁体嵌入转子内部
   - 凸极率 L_d/L_q < 1
   - 可利用磁阻转矩

### 按绕组连接

1. **星形连接 (Y型)**
   - 中点悬空或引出
   - 线电压 = √3 × 相电压
   - 线电流 = 相电流

2. **三角形连接 (Δ型)**
   - 三相首尾相连
   - 线电压 = 相电压
   - 线电流 = √3 × 相电流

## 常见问题

### Q1: BLDC 和 PMSM 的区别？

**答**：术语使用差异：
- **BLDC**：通常指梯形波反电动势，方波控制
- **PMSM**：通常指正弦波反电动势，FOC 控制
- 实际应用中两者常混用

### Q2: 极数和极对数的关系？

**答**：
```
极对数 P = 极数 / 2
电角度 = 机械角度 × P
机械转速 = 电转速 / P
```

例如：12 极电机，P = 6
- 360° 机械转角 = 2160° 电角度

### Q3: 如何提高电机效率？

**答**：
1. 优化换相时机（FOC 控制）
2. 降低铁损（适当磁通密度）
3. 降低铜损（增大线径、降低电阻）
4. 减少机械摩擦
5. 优化驱动波形（SVPWM）
