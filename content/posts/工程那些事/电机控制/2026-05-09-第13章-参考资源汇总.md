## 数据手册速查

### 电机参数测量

| 参数 | 测量方法 | 注意事项 |
|---|---|---|
| 相电阻 | 万用表测量两相 | 阻值小，精度要求高 |
| 相电感 | 电桥测量 | 或使用数据手册 |
| KV 值 | 空载转速/电压 | 需要转速测量工具 |
| 极对数 | 物理数极数 | 极对数 = 极数 / 2 |
| 惯量 | 查数据手册或实测 | 需要专业设备 |

### 控制参数经验值

| 参数 | 典型值 | 调整范围 |
|---|---|---|
| PWM 频率 | 20kHz | 10-50kHz |
| 电流环采样 | 20kHz+ | ≥ PWM 频率 |
| 速度环采样 | 1-5kHz | 电流环/5 |
| 位置环采样 | 100-1kHz | 速度环/5 |

## 公式速查

### 基本公式

```
反电动势：E = k_e × ω
转矩：τ = k_t × I
最大力：F_max = B × I × L
KV 值：KV = 60 / (2π × k_e)
```

### 坐标变换

```
Clarke:
  i_α = i_a
  i_β = (i_b - i_c) / √3

Park:
  i_d = i_α·cos(θ) + i_β·sin(θ)
  i_q = -i_α·sin(θ) + i_β·cos(θ)

逆 Park:
  u_α = u_d·cos(θ) - u_q·sin(θ)
  u_β = u_d·sin(θ) + u_q·cos(θ)
```

### PI 控制器

```
output = Kp × error + Ki × ∫error dt

离散化：
  u[n] = Kp × e[n] + Ki × T_s × Σe[k]
```

## 学习路径建议

### 第一阶段（2-3周）

1. 阅读《电机学》基础章节
2. 学习 SimpleFOC 理论文档
3. 搭建 Arduino + 简单驱动器 + BLDC
4. 实现开环速度控制

### 第二阶段（3-4周）

1. 学习 Clarke/Park 变换
2. 实现 FOC 控制
3. 添加编码器反馈
4. 整定电流环 PID

### 第三阶段（4-6周）

1. 研究三环控制
2. 实现速度环和位置环
3. 学习 SVPWM 原理
4. 迁移到 STM32

### 第四阶段（持续）

1. 研究无传感器 FOC
2. 实现观测器算法
3. 研究多电机协同
4. 产品化开发

## 常见问题汇总

### 硬件问题

| 问题 | 可能原因 | 解决方法 |
|---|---|---|
| 电机不转 | 接线错误、电流限太低 | 检查接线、增加电流限制 |
| 电机抖动 | PID 不当、噪声 | 重新整定 PID、添加滤波 |
| 电机过热 | 电流过大、散热不良 | 降低电流、改善散热 |
| 驱动器烧毁 | 短路、过压 | 检查电路、增加保护 |

### 软件问题

| 问题 | 可能原因 | 解决方法 |
|---|---|---|
| 编译错误 | 头文件未包含、库路径错误 | 检查包含路径 |
| 控制不稳定 | 参数整定不当 | 使用阶跃响应调试 |
| 性能不足 | 采样率不够、算法优化 | 提高采样率、优化代码 |
| 震荡振荡 | 增益太大、反馈延迟 | 降低增益、检查传感器延迟 |
