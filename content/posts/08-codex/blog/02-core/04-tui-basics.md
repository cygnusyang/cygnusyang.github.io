Codex 的终端界面是基于 **Ratatui** 构建的，功能强大且直观。

## 启动 TUI

```bash
codex
```

这会打开一个全屏的终端界面。

## 界面布局

```
┌─────────────────────────────────────────┐
│  Chat History                           │  ← 对话历史
│  User: ...                             │
│  Codex: ...                            │
│  ...                                  │
├─────────────────────────────────────────┤
│  > Your input here...                  │  ← 输入框
├─────────────────────────────────────────┤
│  [Status] Help: ? | Quit: Ctrl+C       │  ← 状态栏
└─────────────────────────────────────────┘
```

## 快捷键

### 基本操作

| 快捷键 | 作用 |
|-------|------|
| `?` | 显示帮助 |
| `Ctrl+C` | 退出 |
| `Enter` | 发送消息 |
| `Up` / `Down` | 浏览历史消息 |

### 编辑

| 快捷键 | 作用 |
|-------|------|
| `Left` / `Right` | 移动光标 |
| `Home` / `End` | 跳到行首/行尾 |
| `Ctrl+A` | 跳到行首 |
| `Ctrl+E` | 跳到行尾 |

## 会话管理

Codex 会自动保存会话历史，下次启动时可以继续之前的对话。

## 本章小结

**一句话记住**：运行 `codex` 进入 TUI，`?` 看帮助，`Ctrl+C` 退出。
